# Arqueria Educativa

Juego hecho con Python y pygame para practicar calculo mental.
Hay que dispararle al blanco que tenga la respuesta correcta.

## Controles

- Flechas ARRIBA/ABAJO: angulo
- ESPACIO: cargar y disparar
- ENTER: continuar
- ESC: menu / salir

## Instalar y correr

En mi maquina pygame normal no me anduvo, asi que uso pygame-ce que es lo mismo pero con algunas correcciones.

```
pip install -r requirements.txt
python Main.py
```
